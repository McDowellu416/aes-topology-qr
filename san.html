What is a SAN?

A Storage Area Network is a dedicated network used to connect servers to shared storage.

Instead of storage being attached to only one machine, a SAN allows storage resources to be centralised and accessed by connected servers.

Benefits of a SAN

  • Provides centralised storage for multiple servers.
  • Can offer high-speed access to shared storage.
  • Supports backup, resilience and storage management.
  • Can make storage easier to scale and manage.

Limitations of a SAN

  • More complex than simple direct-attached storage.
  • Can be expensive to install and maintain.
  • Requires specialist switches, cabling and configuration.
  • Poor design can create performance or availability issues.
